
# Project Title
GroundUP Landing Page

## Description
The GroundUP landing page is the result of a Front End Tools & Portfolio course assignment from learning institution NobleDesktop.
The purpose of the assignment was to apply the course concepts namely html, CSS, and JS for a little extra credit.  A figma file was provided
displaying a coffee shop landing page and the assignment was to write the code in html and CSS to replicate all aspects of the landing page.

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/combatbrownie/GroundUP-Landing-Page.git
    ```

2. Navigate to the project directory:
    ```bash
    cd GroundUP-Landing-Page
    ```

3. Install dependencies:
    ```bash
    npm install
    ```
